Tag Info
<!DOCTYPE> Depicts about html version
<body> All the content is written inside it.
<div> Specifies a section in a document
<title> Specifies a title for the document
<meta> Specifies metadata about an HTML document
<nav> Specifies navigation links
<button> Specifies a clickable button
<!--...--> For writing comments
<a> Insert a hyperlink
<article> Specifies an article
<b> Used for bold text
<br> Specifies a single line break
<caption> Inserts a table caption
<code> Specifies a piece of computer code
<del> Specifies deleted text
<dd> Specifies a definition description
<details> Specifies additional details that the user can view or hide
<dialog> Specifies a dialog box or window
<em> Specifies emphasized text
<embed> Specifies a container for an external (non-HTML) application
<font> Specifies font, color, and size for text.    (Not supported in HTML5. Use CSS instead.)
<footer> Specifies a footer for a document or section
<form> Specifies an HTML form for user input
<frame> Specifies a window (a frame) in a frameset.    (Not supported in HTML5. Use CSS instead.)
<frameset> Specifies a set of frames.    (Not supported in HTML5. Use CSS instead.)
<h1> to <h6> Specifies HTML headings
<head> Specifies information about the document
<header> Specifies a header for a document or section
<hr> Specifies a thematic change in the content
<html> Specifies the root of an HTML document
<i> Specifies a part of text in an alternate voice or mood
<iframe> Specifies an inline frame
<img> Specifies an image
<label> Specifies a label for an <input> element
<li> Specifies a list item
<link> Specifies the relationship between a document and an external resource (most used to link to style sheets)
<main> Specifies the main content of a document
<map> Specifies a client-side image-map
<mark> Specifies marked/highlighted text
<noframes> Specifies an alternate content for users that do not support frames.     (Not supported in HTML5. Use CSS instead.)
<noscript> Specifies an alternate content for users that do not support client-side scripts
<ol> Specifies an ordered list
<option> Specifies an option in a drop-down list
<p> Specifies a paragraph
<pre> Specifies preformatted text
<script> Specifies a client-side script
<section> Specifies a section in a document
<select> Specifies a drop-down list
<small> Specifies smaller text
<span> Specifies a section in a document
<strong> Specifies important text
<style> Specifies style information for a document
<sub> Specifies subscripted text
<sup> Specifies superscripted text
<table> Specifies a table
<tbody> Groups the body content in a table
<td> Specifies a cell in a table
<tfoot> Groups the footer content in a table
<th> Specifies a header cell in a table
<thead> Groups the header content in a table
<tr> Specifies a row in a table
<u> Specifies text that should be stylistically different from normal text
<ul> Specifies an unordered list
